## Getting Started

Для запуска потребуется `python=3.10.2`

### Installing

1. Склонируйте репозиторий и запустите:

    ```bash
    python main.py
    ```

## Examples

```bash
>> python main.py
>> 3 + 4 * ( 2 - 1 )
>> 3421-*+
```

```bash
>> python main.py
>> 5 + 3 * 7
>> 537*+
```

```bash
>> python main.py
>> 3 * ( sin ( 4 ) - 1 ) + 4 * ( cos ( 2 ) * 7 )
>> 341-sin*427*cos*+
```
